
## Project setup
1. Fork the repo
2. `git clone https://github.com/<your_username>/rTN_repo.git`
3. `cd rTN_repo`
4. `npm install` 

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
